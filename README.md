## Overview

At-Tahirah Library is a clean, personal library web app built with HTML, CSS, and vanilla JavaScript. It allows users to add book entries, manage a collection, and update book status using an intuitive card-based interface.

## Live Demo

Published with GitHub Pages:

https://maryam-20.github.io/atTahira-library/

## Features

- Add new book entries with title, author, page count, and genre.
- Display book cards in a responsive collection layout.
- Toggle between "Read" and "Unread" status.
- Delete books from the collection.
- Clean sidebar navigation and search input styling.

## How to Use

1. Open `index.html` in your browser or serve the project from a local web server.
2. Fill in the book form with title, author, pages, and genre.
3. Click the `+ Add Entry` button to add the book to your collection.
4. Use the `Mark as Read` / `Mark as Unread` button to update the book status.
5. Delete unwanted entries with the `Delete` button.

## Project Structure

- `index.html` — main layout and form.
- `style.css` — app styling and responsive layout.
- `scripts.js` — book management logic and DOM rendering.
- `README.md` — project documentation.

## Notes

This project is a front-end prototype for a personal archive system. Future improvements can include persistent storage with `localStorage`, enhanced search, and filtering by genre or read status.
